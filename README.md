## 有關 github
1. 沒有帳號的話先去申請一個
2. 把 github ID 跟我說，像我是 HardworkingSnowman
3. 打開 command line 或是 terminal
```
# 先 cd 到想要存放的資料夾
cd "PATH"

# 把 project 複製到資料夾中
git clone https://github.com/HardworkingSnowman/nctu_ML.git
```
4. 以後當做完一部份，想要更新到 github 上面時
```
sudo git add --all
sudo git commit -m "裡面寫改了什麼，寫什麼都可以"
sudo git push
```
